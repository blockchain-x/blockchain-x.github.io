从零到壹学习EOS - cleos 命令

发表于 2018-05-24

查看 cleos 命令

version

  version 中cleos version client 返回客户端的版本信息

create

  cleos create 主要包含 key 和 account两个子命令

cleos create key

  创建一对新的公钥和私钥

cleos create account

get

cleos get info

    获取当前区块链的信息

cleos get block

   根据 block的块号 ( block_num ) 和 id 进行查询。
   cleos get block 110 根据区块的 block_num 进行查询。

cleos get block 0000006eb3d79a21d0e4980212ed69a4e4e70b9576958f6872b698a43d0655b3 根据区块的 id 进行查询

cleos get account

    获取相应的账号信息

cleos get code

   返回指定账户名的代码信息

cleos get table

   返回指定表信息

cleos get currency

cleos get currency balance

cleos get currency stats

cleos get accounts

  返回检查node的配置，加入一项plugin = eosio::history_api_plugin 发现node无法正常启动，这个还无法解决。TODO

cleos get servants

cleos get transactions

set

cleos set contract

wallet

cleos wallet create

    通过 -n 指定相应的钱包的名字

cleos wallet open

    通过 -n 打开相应的钱包

cleos wallet lock

   通过 -n 打开相应的钱包

cleos wallet lock_all

cleos wallet unlock

cleos wallet import

    将 私钥 导入到钱包中

cleos wallet list

cleos wallet keys

   返回所有解锁钱包含有的公/私钥对

push

    cleos push action 推送一条操作事务
